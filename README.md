# Student Performance Prediction Using Linear Regression

## Project Overview

This project demonstrates an end-to-end Machine Learning workflow for predicting student exam scores based on study hours. The project covers essential data preprocessing techniques, outlier detection and treatment, feature engineering, model training, evaluation, and visualization.

The objective is to understand how data preparation impacts model performance and how Linear Regression can be used to identify relationships between variables and generate predictions.

## Project Workflow

### 1. Data Exploration

The dataset was explored using:

* head()
* tail()
* info()
* describe()

This helped understand the structure, data types, and statistical properties of the dataset.

### 2. Data Quality Assessment

Performed checks for:

* Missing values
* Duplicate records
* Unique category values

These steps are important to ensure data reliability before model training.

### 3. Feature Engineering

Applied One-Hot Encoding to transform categorical variables into a machine-learning-friendly format.

### 4. Outlier Analysis

Visualized data using Boxplots and detected extreme values using the Interquartile Range (IQR) method.

The identified outliers were removed to improve data quality and model performance.

### 5. Model Development

A Linear Regression model was trained to learn the relationship between study hours and exam scores.

### 6. Model Evaluation

The trained model was evaluated using:

* Actual vs Predicted Comparison
* R² Score
* Error Metrics
* Visualization of Regression Line

### 7. Visualization

Created a scatter plot with a regression line to clearly illustrate the relationship between study hours and exam scores.
